# Enterprise Windows Server Home Lab

## Overview

This repository documents my hands-on Windows Server home lab built using VMware Workstation Pro. The project simulates an enterprise IT environment to develop practical experience with Windows Server administration, Active Directory, networking, DNS, DHCP, Group Policy, PowerShell, and IT support troubleshooting.

The lab is developed using a combination of Microsoft Learn documentation, VMware documentation, industry-recognized Windows Server and Active Directory training resources, and hands-on practice. I follow guided tutorials to learn new concepts and then reinforce those skills by recreating the configurations within my own virtual environment.

To strengthen my troubleshooting skills, I investigate, diagnose, and resolve realistic IT support scenarios sourced from training materials, technical documentation, community discussions, and self-created break/fix exercises. Every major configuration and troubleshooting exercise is documented with the investigation process, root cause, resolution, and lessons learned.

The objective of this project is to build a complete enterprise Windows Server environment while developing the practical skills expected of an entry-level IT Support Specialist, Help Desk Technician, or Systems Administrator.
